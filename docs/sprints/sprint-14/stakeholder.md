# Sprint 14 — Stakeholder Brief: Session Data Export & Visual Reports

## User Problem
Users have no way to extract their focus data for use in external tools (spreadsheets, reports)
or to share productivity summaries with managers, coaches, or accountability partners.
All data is locked inside the app with no export path.

## Feature: Session Export & Reports
**Story Points: 8**

### Key Needs
1. **CSV Export** — Export all sessions or a filtered date range as a CSV file via a native save dialog
2. **In-app Report** — Generate and view a human-readable productivity report (weekly/monthly)
3. **Report Contents**
   - Total sessions
   - Total focus hours
   - Average session length
   - Best day (most focused)
   - Current streak info
   - Per-day breakdown with mini bar chart
4. **Copy to Clipboard** — Copy the formatted report text for pasting into emails/docs

### Success Criteria
- User can export data without technical knowledge
- Report is readable and useful without any formatting tool
- Works offline, no external dependencies
